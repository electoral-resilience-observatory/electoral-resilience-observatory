# Electoral Resilience Observatory

A standalone website for the Electoral Resilience Observatory.

## Description

The Electoral Resilience Observatory is a research and policy platform for analyzing electoral threats, institutional responses, organizational learning, and democratic resilience across election reports.

## Sections

- Home
- About
- Dashboards
- Reports
- Methods
- Data
- Contact

## Deployment

This site can be hosted with GitHub Pages.

1. Create a new GitHub repository.
2. Upload `index.html`, `style.css`, and `README.md`.
3. Go to repository Settings.
4. Open Pages.
5. Select the main branch as the source.
6. Save and publish.
